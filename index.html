My Work

Where's Waldo game implemention using Ruby on Rails on the server and jquery on the client. The server contains the position of the characters you are looking for. When the user selects a character among the crowd, the client sends an api call to the server to check if the user has found a character. The server also holds the high scores for each level. This project is hosted on Heroku.

Sketchpad game implememnted using jquery. When the user places the mouse pointer over a box it changes color. The grid can be changed up to a max of 64x64 grid. This project is hosted on Github Pages.
